<task type="auto">
  <name>Task 3: Create transaction classifier function</name>
  <files>src/classifiers/transaction.classifier.ts</files>
  <action>
Create classifier with two exported functions:

1. `classifyTransaction(tx: ChainTransactionWithLogs, sourceChainId: number): ClassifiedTransaction`
   - Iterate through tx.logEvents
   - Try decoding each with viem's `decodeEventLog({ abi: LIFI_EVENTS_ABI, data, topics, strict: false })`
   - If `LiFiTransferStarted` found:
     - Extract destinationChainId from decoded.args.bridgeData.destinationChainId
     - type = destinationChainId !== sourceChainId ? 'bridge' : 'swap'
   - If `LiFiGenericSwapCompleted` found:
     - type = 'swap', destinationChainId = null
   - If no LiFi events decoded (or all decoding fails):
     - Default to type = 'swap', destinationChainId = null (conservative per CONTEXT.md)
   - Use try/catch around decodeEventLog, continue to next log on error

2. `classifyAllTransactions(rawTransactions: ChainTransactionWithLogs[]): ClassificationResult`
   - Filter out transactions where !tx.successful (failed/reverted excluded per CONTEXT.md)
   - For each remaining transaction, call classifyTransaction
   - Catch errors per-transaction and add to errors array
   - Return { classified, errors }

Import viem's decodeEventLog:
```typescript
import { decodeEventLog } from 'viem';
```

Cast topics array for viem:
```typescript
topics: log.raw_log_topics as [`0x${string}`, ...`0x${string}`[]]
```
